What other members think about my ideas:

My teammates and I both agree that I have given a detailed idea yet ambitious and unrealistic one. My project idea has listed quite detail functions of the autonomous robot but the way how to do create and build is missing. The reason for it is because I am lack of knowledge about create a robot, also the way to make it run autonomously. It a common sense for a freshman like me, also other teammates. Because of that, my project idea is likely a professional scientist project than a project for students. It is advanced and complicated because our team needs to work both on software and hardware. For software, we need to have deep understanding about Python, as well as Arduino IDE and Raspbian OS to perform closed-loop control system for the robot to make decisions on its own. For hardware, we need to have some knowledge about small computing devices, especially how to build Arduinos and Raspberry Pi boards. Time limit is also a difficulty for us because for a project that is complicated like that, there should be at least 2 to 3 months to complete it, it is impossible for us to complete it in 3 weeks, not to mention we also have assignments that need to be completed for other courses. Also, it might cost lots of money to buy the hardware in order to construct the robot, which is not necessary. Not for the content of the project idea, but those feedbacks make me realize that I am so ambitious and have put myself too much. To be honest, I really like to build an autonomous one day but not right now.
